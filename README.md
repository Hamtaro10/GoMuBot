# GoMuBot
## Discord Bot With AIO (All In One) Features
GoMuBot adalah bot Discord yang dibangun menggunakan Python, Pomice (Wrapper), dan Lavalink (Music Player). Bot ini adalah bot serbaguna (All In One) yang memiliki fitur pemutaran lagu dari YouTube dengan fitur antrian, pemutaran otomatis, looping, dan pencarian menggunakan YouTube Data API v3.

## Features

Berikut adalah fitur fitur yang sementara dimiliki oleh GoMu bot :

- Play music (skip, join, play, loop, queue, skip, stop)
- Fitur utilitas untuk user (Set AFK, Check ping)
