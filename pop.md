# Privacy Policy – GoMu Bot

**Terakhir diperbarui:** 29 Juli 2025

GoMu Bot (“kami”, “Bot”) menghargai privasi pengguna dan berkomitmen untuk melindungi informasi yang dikumpulkan saat penggunaan layanan Bot kami di Discord.

---

## 1. Informasi yang Dikumpulkan

Kami mengumpulkan data **terbatas** yang diperlukan agar Bot dapat berfungsi dengan baik, termasuk:

- ID pengguna dan server (Guild ID, User ID, Channel ID)
- Perintah yang Anda jalankan (misalnya: `g!play`, `g!afk`)
- Informasi suara untuk fitur pemutaran musik (tanpa merekam audio)

---

## 2. Cara Penggunaan Data

Data tersebut digunakan untuk:
- Memberikan layanan seperti musik, pengecekan status, dan fitur AFK
- Memelihara performa dan stabilitas Bot
- Menyimpan konfigurasi server (seperti preferensi AFK atau antrian musik)

Kami **tidak menjual, menyewakan, atau membagikan data** ke pihak ketiga.

---

## 3. Penyimpanan Data

- Sebagian data (seperti status AFK, queue lagu) mungkin disimpan sementara selama sesi aktif.
- Data disimpan secara minimal dan hanya selama dibutuhkan untuk menjalankan fitur.

---

## 4. Hak Anda

Sebagai pengguna:
- Anda berhak meminta penghapusan data dari server kami (jika berlaku)
- Anda dapat menghapus Bot dari server kapan saja untuk menghentikan pengumpulan data

---

## 5. Keamanan

Kami berusaha menjaga keamanan data dengan metode standar pengamanan aplikasi. Namun, tidak ada sistem yang 100% aman dari pelanggaran.

---

## 6. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan ini sewaktu-waktu. Perubahan signifikan akan diumumkan melalui GitHub atau Discord server resmi (jika tersedia).
